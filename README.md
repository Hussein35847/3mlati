# 💱 3mlati (عملاتي)

> **Smart Currency Converter for the Middle East**

3mlati is a modern, responsive web application that allows users to convert currencies quickly and efficiently.
Built with a focus on simplicity and real-world usability, it supports both global and regional currencies.

---

## 🌍 Supported Currencies

* 🇯🇴 Jordanian Dinar (JOD)
* 🇺🇸 US Dollar (USD)
* 🇪🇺 Euro (EUR)
* 🇸🇾 Syrian Pound (SYR)
* 🇮🇶 Iraqi Dinar (IRQ)
* 🇹🇷 Turkish Lira (TRY)

---

## ⚡ Features

* 💱 Real-time currency conversion
* 🔄 Swap currencies instantly
* 📱 Fully responsive (mobile & desktop)
* 🎯 Clean and intuitive UI
* ⚠️ Input validation & error handling
* ⏳ Loading states for better UX

---

## 🎨 UI & Design

* Minimal, modern fintech-inspired interface
* Focused on speed and usability
* Smooth interactions and clean layout

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)
